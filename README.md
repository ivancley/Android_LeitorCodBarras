Projeto Android Leitor de Código de Barras
==========================================
Projeto utilizando a biblioteca ZXing para realizar leitura de código de barra

Links 
Página do projeto ZXing (https://github.com/zxing/zxing)
ZXing Core 3.2.1 (http://mvnrepository.com/artifact/com.google.zxing/core/3.2.0)
Página para testes de leitura (http://www.barcode-generator.org)

Observações
O Projeto deve ser compilado em Java 1.7
API 9 mínima requerida
