Projeto Android Leitor de Código de Barras
==========================================
Projeto utilizando a biblioteca ZXing para realizar leitura de código de barra

<p>Links </p>
<p>Página do projeto ZXing (https://github.com/zxing/zxing)</p>
<p>ZXing Core 3.2.1 (http://mvnrepository.com/artifact/com.google.zxing/core/3.2.0)</p>
<p>Página para testes de leitura (http://www.barcode-generator.org)</p>
<br />
<p>Observações</p>
<p>O Projeto deve ser compilado em Java 1.7</p>
<p>API 9 mínima requerida</p>
